# Android 冷啟動藍屏/白屏問題 - 完整解決方案

## 🔍 問題描述

**現象：** 在顯示 SplashActivity 之前，會先出現一個藍色的畫面（約 0.5-1 秒）

**原因：** 這是 Android 的冷啟動預覽窗口（Preview Window）

---

## 📖 技術背景

### Android 應用啟動流程

```
使用者點擊應用圖示
    ↓
系統顯示預覽窗口 (Preview Window)  ← 這裡出現藍色畫面！
    ↓
應用程序進程啟動
    ↓
Application.onCreate() 執行
    ↓
SplashActivity 創建
    ↓
SplashActivity.onCreate() 執行
    ↓
布局渲染完成
    ↓
使用者看到 SplashActivity
```

### 為什麼會有預覽窗口？

- **目的：** 讓用戶立即得到視覺反饋，提升體驗
- **來源：** 使用 Activity 的 `android:theme` 中的 `windowBackground`
- **問題：** 如果 `windowBackground` 與實際 Activity 布局不一致，會產生閃爍感

---

## 🎯 原始配置（有問題）

### colors.xml
```xml
<color name="colorPrimary">#2196F3</color>  <!-- 藍色 -->
```

### themes.xml
```xml
<style name="Theme.Splash" parent="...">
    <item name="android:windowBackground">@color/colorPrimary</item>
    <!-- ↑ 這裡導致了藍色畫面 -->
</style>
```

### 結果
```
用戶看到：藍色畫面 → SplashActivity（白色文字+藍色背景）
          ↑ 這是預覽窗口
```

---

## ✅ 解決方案

### 方案 1：使用自訂啟動畫面 Drawable（推薦）⭐

**原理：** 讓預覽窗口和 SplashActivity 看起來一致

#### 步驟 1：創建 splash_background.xml

```xml
<!-- app/src/main/res/drawable/splash_background.xml -->
<layer-list xmlns:android="http://schemas.android.com/apk/res/android"
    android:opacity="opaque">
    
    <!-- 背景色 - 與 SplashActivity 相同 -->
    <item android:drawable="@color/colorPrimary" />
    
    <!-- 中央 Logo -->
    <item>
        <bitmap
            android:gravity="center"
            android:src="@mipmap/ic_launcher" />
    </item>
    
</layer-list>
```

#### 步驟 2：更新 themes.xml

```xml
<style name="Theme.Splash" parent="Theme.AppCompat.Light.NoActionBar">
    <!-- 從純色改為 drawable -->
    <item name="android:windowBackground">@drawable/splash_background</item>
    <item name="android:windowFullscreen">true</item>
    <item name="android:windowContentOverlay">@null</item>
    <item name="android:windowIsTranslucent">false</item>
    <item name="android:windowNoTitle">true</item>
    <item name="android:windowActionBar">false</item>
</style>
```

#### 效果
```
用戶看到：啟動畫面（Logo+藍色背景）→ SplashActivity（幾乎無縫銜接）
          ↑ 預覽窗口                    ↑ 實際 Activity
```

---

### 方案 2：完全透明（不推薦）

**原理：** 讓預覽窗口透明

```xml
<style name="Theme.Splash" parent="...">
    <item name="android:windowIsTranslucent">true</item>
    <item name="android:windowBackground">@android:color/transparent</item>
</style>
```

**缺點：**
- ❌ 啟動速度會稍微變慢
- ❌ 用戶點擊後沒有立即視覺反饋
- ❌ 不符合 Material Design 指南

---

### 方案 3：使用 Android 12+ Splash Screen API（最新）

**適用於：** Android 12 (API 31) 及以上

```xml
<!-- themes.xml (v31) -->
<style name="Theme.Splash" parent="Theme.SplashScreen">
    <item name="windowSplashScreenBackground">@color/colorPrimary</item>
    <item name="windowSplashScreenAnimatedIcon">@mipmap/ic_launcher</item>
    <item name="windowSplashScreenAnimationDuration">200</item>
    <item name="postSplashScreenTheme">@style/Theme.Functiontest</item>
</style>
```

**需要：**
```gradle
dependencies {
    implementation 'androidx.core:core-splashscreen:1.0.1'
}
```

```kotlin
// SplashActivity.kt
override fun onCreate(savedInstanceState: Bundle?) {
    val splashScreen = installSplashScreen()
    super.onCreate(savedInstanceState)
    // ...
}
```

### 創建更精美的啟動畫面

#### splash_background_advanced.xml
```xml
<layer-list xmlns:android="http://schemas.android.com/apk/res/android"
    android:opacity="opaque">
    
    <!-- 漸變背景 -->
    <item>
        <shape android:shape="rectangle">
            <gradient
                android:angle="135"
                android:centerColor="@color/colorPrimary"
                android:endColor="@color/colorPrimaryDark"
                android:startColor="#42A5F5"
                android:type="linear" />
        </shape>
    </item>
    
    <!-- Logo（固定大小） -->
    <item
        android:width="120dp"
        android:height="120dp"
        android:gravity="center">
        <bitmap
            android:gravity="center"
            android:src="@mipmap/ic_launcher" />
    </item>

```

## 💡 最佳實踐

### 1. 啟動畫面應該簡潔
- ✅ Logo 或品牌標識
- ✅ 純色或簡單漸變背景
- ❌ 不要放太多元素
- ❌ 不要放複雜動畫（影響性能）

### 2. 顏色一致性
- 預覽窗口顏色應與 SplashActivity 背景一致
- 避免明顯的顏色跳變

### 3. 時間控制
- 啟動畫面不宜太長（1-2 秒為佳）
- 權限檢查可以在後台進行

### 4. 不同 Android 版本
- Android 12+ 使用 Splash Screen API
- Android 12 以下使用自訂主題

## 🆘 常見問題

### Q1: 修改後還是有藍色閃爍？

**A:** 確認以下事項：
1. AndroidManifest.xml 中 SplashActivity 的 `android:theme="@style/Theme.Splash"`
2. themes.xml 中已更新為 `@drawable/splash_background`
3. 重新安裝應用（不是熱更新）

### Q2: Logo 顯示太大或太小？

**A:** 使用進階版 `splash_background_advanced.xml`，調整 `android:width` 和 `android:height`

### Q3: 想使用漸變背景？

**A:** 在 `splash_background_advanced.xml` 中取消註解 `<gradient>` 部分

### Q4: Android 12+ 設備顯示不正常？

**A:** Android 12+ 有新的 Splash Screen 系統，考慮使用 `androidx.core:core-splashscreen` 庫

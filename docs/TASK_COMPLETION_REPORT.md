## ⚠️ 注意事项

### 开发注意事项

1. **Core 模块是 Submodule**
   - 如果修改 core 中的代码，需要双重提交
   - 先在 core 仓库提交，再在主项目更新 submodule 指针

2. **设备检测依赖**
   - 必须确保 HardwareManager 在 SplashActivity 中正确初始化
   - 如果初始化失败，所有设备选单都会禁用

3. **真机测试必要性**
   - 模拟器只能测试 DummyHelper 场景
   - PAX 和 Castles 功能必须在真机上测试

### 已知限制

1. **静态检测**
   - 设备类型在 initDrawerData() 时检测，不支持运行时切换
   - 如需切换设备，需要重启应用

2. **视觉效果**
   - 禁用效果使用 alpha 和 color，不支持自定义样式
   - 可能需要根据主题颜色调整

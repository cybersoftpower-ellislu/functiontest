## 📊 技术架构总结

### 1. SplashActivity（启动画面）

**架构：** MVVM
- **View:** `SplashActivity.java` + `activity_splash.xml`
- **ViewModel:** `SplashViewModel.java`
- **Binding:** DataBinding（`ActivitySplashBinding`）

**功能：**
- ✅ 显示启动画面（1.5 秒）
- ✅ 检查并请求 5 项权限
- ✅ 自动跳转到 MainActivity
- ✅ 禁用返回键

**特点：**
- 继承 `BaseActivity<ActivitySplashBinding, SplashViewModel>`
- 使用 `SingleLiveEvent` 处理事件
- 实现最短显示时间机制
- 权限拒绝对话框

### 2. 冷启动优化

**主题配置：** `Theme.Splash`
```xml
<style name="Theme.Splash">
    <item name="android:windowBackground">@drawable/splash_background</item>
    <item name="android:windowFullscreen">true</item>
    ...
</style>
```

**启动画面 Drawable：** `splash_background.xml`
```xml
<layer-list>
    <item>
        <shape android:shape="rectangle">
            <solid android:color="@color/colorPrimary" />
        </shape>
    </item>
</layer-list>
```

**效果：**
- ✅ 预览窗口显示蓝色纯色背景
- ✅ 与 SplashActivity 颜色一致
- ✅ 无视觉跳变，无闪烁

### 3. 权限管理

**位置：** SplashActivity
**权限清单：**
1. `CAMERA` - 相机
2. `READ_EXTERNAL_STORAGE` - 读取储存
3. `WRITE_EXTERNAL_STORAGE` - 写入储存
4. `ACCESS_FINE_LOCATION` - 精确位置
5. `ACCESS_COARSE_LOCATION` - 粗略位置

**流程：**
```
SplashActivity.onCreate()
    ↓
checkAndRequestPermissions()
    ↓
onRequestPermissionsResult()
    ↓
viewModel.onPermissionResult()
    ↓
permissionCheckCompleteEvent
    ↓
navigateToMainWithDelay()
    ↓
navigateToMain()
```

### 4. MainActivity

**架构：** MVVM
**功能：**
- ✅ 两层 Drawer 导航
- ✅ 功能测试按钮
- ✅ 离开应用功能
- ✅ 所有权限已在 SplashActivity 检查完成

### 2. MVVM 架构模式

**优点：**
- ✅ 代码分离清晰
- ✅ ViewModel 可重用
- ✅ 生命周期管理自动
- ✅ 数据绑定简化代码

**实现：**
```java
// Activity
public class SplashActivity extends BaseActivity<ActivitySplashBinding, SplashViewModel> {
    @Override
    public int bindViewModelId() {
        return BR.viewModel;
    }
    
    @Override
    public int bindContentView(Bundle savedInstanceState) {
        return R.layout.activity_splash;
    }
}

// ViewModel
public class SplashViewModel extends BaseViewModel {
    public SingleLiveEvent<Boolean> permissionCheckCompleteEvent = new SingleLiveEvent<>();
    public SingleLiveEvent<Void> navigateToMainEvent = new SingleLiveEvent<>();
    
    public void onPermissionResult(boolean allGranted) {
        permissionCheckCompleteEvent.setValue(allGranted);
    }
}
```

### 3. 冷启动优化

**关键：** 预览窗口与 Activity 视觉一致

**实现：**
1. 创建 `splash_background.xml` drawable
2. 在 `Theme.Splash` 中设置为 `windowBackground`
3. 使用与 Activity 相同的背景色
4. 效果：无缝过渡，无闪烁

### 4. 权限管理

**最佳实践：**
- ✅ 在 SplashActivity 统一检查
- ✅ 使用 ViewModel 处理结果
- ✅ 提供用户友好的说明对话框

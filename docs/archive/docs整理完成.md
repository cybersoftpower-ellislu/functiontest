## 🎯 使用指南

### 查找文檔
1. **快速了解專案**: 閱讀根目錄 `README.md`
2. **了解架構**: 閱讀 `AGENTS.md`
3. **查看開發文檔**: 進入 `docs/` 目錄或閱讀 `docs/README.md`

### 添加新文檔
當創建新的技術文檔時：
1. 將文檔放置在 `docs/` 目錄
2. 在 `docs/README.md` 中添加索引
3. 如需要，在專案 `README.md` 中添加連結

### 文檔命名
- 中文或英文皆可
- 使用描述性檔名
- 建議格式: `功能_類型.md` 或 `FEATURE_TYPE.md`

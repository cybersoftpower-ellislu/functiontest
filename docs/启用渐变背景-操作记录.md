## 🎨 自定义选项

### 如果想调整渐变效果

#### 调整渐变角度

**当前：** 135° (左上到右下)

**可选方向：**
```xml
android:angle="0"    → 从左到右 (水平)
android:angle="45"   → 左上到右下偏右
android:angle="90"   → 从上到下 (垂直)
android:angle="135"  → 左上到右下 (对角线) ✅ 当前
android:angle="180"  → 从右到左
android:angle="270"  → 从下到上
```

#### 调整颜色

**修改 `splash_background_advanced.xml`：**

```xml
<gradient
    android:angle="135"
    android:startColor="#你的起点颜色"
    android:centerColor="#你的中点颜色"
    android:endColor="#你的终点颜色"
    android:type="linear" />
```

**建议保持与主题色相协调！**

## 📝 技术细节

### Gradient 属性说明

```xml
<gradient
    android:angle="135"              <!-- 渐变角度 -->
    android:centerColor="#2196F3"    <!-- 中间颜色 -->
    android:endColor="#1976D2"       <!-- 结束颜色 -->
    android:startColor="#42A5F5"     <!-- 起始颜色 -->
    android:type="linear" />         <!-- 渐变类型：线性 -->
```

**其他可选属性：**
- `android:centerX` / `centerY` - 渐变中心位置（0 到 1）
- `android:gradientRadius` - 径向渐变的半径
- `android:type="radial"` - 径向渐变（从中心向外）
- `android:type="sweep"` - 扫描渐变（圆形旋转）
